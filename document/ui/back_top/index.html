回到顶部 #

该组件常用于当页面比较长时回到顶部。

基础用法 #

当页面滑动到一定位置出现。

>{}
<template>
    <div>向下滚动页面出现回到顶部组件。</div>
    <as-top :visibility-height="10" />
</template>

2
3
4

自定义内容 #

在40px * 40px的区域内, 其中的内容可自定义。

>{}
<template>
    <div>向下滚动页面出现回到顶部组件（自定义样式）。</div>
    <as-top :bottom="100" :visibility-height="10">
        <div
            style="
                height: 100%;
                width: 100%;
                background-color: #f2f5f6;
                box-shadow: 0 0 6px rgba(0, 0, 0, 0.12);
                text-align: center;
                line-height: 40px;
                color: #1989fa;
            "
        >
            UP
        </div>
    </as-top>
</template>
