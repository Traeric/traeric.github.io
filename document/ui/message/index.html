Message消息提示 #

该组件常用于在添加、修改、删除等操作后的提示

基础用法 #

Message注册了一个全局$as_message方法用于调用，也可以直接导入AsMessage模块使用。 可以接收一个字符串或一个VNode作为参数。

<template> <as-button type="primary" @click="messageTip">测试</as-button> <as-button type="warning" @click="messageVNode">VNode消息基础用法</as-button> </template> <script setup> import {AsMessage} from "awesome-slider-auth"; import { h } from "vue"; function messageTip() { AsMessage("Message提示基础用法"); } function messageVNode() { AsMessage(h('p', null, [ h('span', null, 'Message消息支持 '), h('i', { style: 'color: teal' }, 'VNode'), ])); } </script>

不同风格 #

除了默认的风格，Message还支持Element Plus风格，设置elementLike为true即可。

<template> <as-button type="warning" @click="messageTip">Message消息基础用法</as-button> <as-button type="negative" @click="messageVNode">VNode消息基础用法</as-button> </template> <script setup> import {AsMessage} from "awesome-slider-auth"; import { h } from "vue"; function messageTip() { AsMessage({ message: "Message提示基础用法", elementLike: true }); } function messageVNode() { AsMessage({ message: h('p', null, [ h('span', null, 'Message消息支持 '), h('i', { style: 'color: teal' }, 'VNode'), ]), elementLike: true }); } </script>

不同类型 #

Message支持四种类型，分别是info、success、warn、error，通过type设置即可。

<template> <as-button type="primary" @click="messageInfo">Info消息</as-button> <as-button type="positive" @click="messageSuccess">Success消息</as-button> <as-button type="warning" @click="messageWarn">Warn消息</as-button> <as-button type="negative" @click="messageError">Error消息</as-button> <as-button type="primary" @click="messageElementInfo">eleemnt Info消息</as-button> <as-button type="positive" @click="messageElementSuccess">element Success消息</as-button> <as-button type="warning" @click="messageElementWarn">element Warn消息</as-button> <as-button type="negative" @click="messageElementError">element Error消息</as-button> </template> <script setup> import {AsMessage} from "awesome-slider-auth"; function messageInfo() { AsMessage({ message: "Info提示基础用法", type: "info" }); } function messageSuccess() { AsMessage({ message: "Success提示基础用法", type: "success" }); } function messageWarn() { AsMessage({ message: "Warn提示基础用法", type: "warn" }); } function messageError() { AsMessage({ message: "Error提示基础用法", type: "error" }); } function messageElementInfo() { AsMessage({ message: "Info提示基础用法", type: "info", elementLike: true, }); } function messageElementSuccess() { AsMessage({ message: "Success提示基础用法", type: "success", elementLike: true }); } function messageElementWarn() { AsMessage({ message: "Warn提示基础用法", type: "warn", elementLike: true }); } function messageElementError() { AsMessage({ message: "Error提示基础用法", type: "error", elementLike: true }); } </script>

关闭时间和关闭回调 #

Message的展示时间可以由duration控制，单位为ms，此外，在关闭时会触发一个回调函数onClose。

<template> <as-button type="primary" @click="messageDuration">设置关闭时间</as-button> <as-button type="positive" @click="messageClose">关闭时触发回调</as-button> </template> <script setup> import {AsMessage} from "awesome-slider-auth"; function messageDuration() { AsMessage({ message: "设置关闭时间为1秒", duration: 1000 }); } function messageClose() { AsMessage({ message: '触发关闭回调', duration: 3000, onClose: () => { AsMessage("关闭回调触发成功！"); } }); } </script>
